About the Shell Script Course 

The course covers Shell Script from the basics to advanced topics, following a hands-on and clear teaching style.

    --> Beginner Module

* Shell navigation
* Variable and types
* Operators and tests 
* Standard input and outputs 
* Conditional structures
* Loops
* How the script run in a sub-process

    --> Intermediate Module

* Functions and modularization
* File and directory manipulation
* Text processing and filters 
* Signals and processes
* Arrays
* Execution control and best practices

    ---> Advanced Module

* Production-grade scripting
* Structured logging
* Error handling and validation
* Defensive programming
* Developing real, reusable Sysadmin tolls
* Creating ultilities used in real-world environments

    PURPOSE

The goal is to train the student to:

* Write professional, clean, safe and portable scripts
* Automate tasks in linux environments
* Solve real problems in companies
* Increase SysAdmin-level technical maturity
* Build a solid Shell Script-based portfolio
